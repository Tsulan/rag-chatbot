Тест USM Chat Widget

Инструкция по использованию:

Чтобы протестировать виджет на любом сайте:

  1. Откройте сайт, на котором хотите протестировать виджет
  2. Откройте Инструменты разработчика (F12)
  3. Перейдите на вкладку Console
  4. Скопируйте и вставьте следующий код:
fetch('http://localhost:3000/widget.js').then(r => r.text()).then(code => eval(code))

Или чтобы загрузить напрямую из браузера:

javascript:(function(){fetch('http://localhost:3000/widget.js').then(r=>r.text()).then(eval)})()

Этот виджет появится в правом нижнем углу страницы и позволит пользователям взаимодействовать с ассистентом USM.

Возможности:

  • Современный и адаптивный дизайн
  • Плавные анимации
  • Поддержка истории чата
  • Индикаторы загрузки
  • Обработка ошибок
